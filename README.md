Projeto de prática Scraper

Tratamento de Exceções de Dados: O código limpa textos e converte números

Interface Responsiva e Temática: Uma galeria de itens que respeita o visual do jogo, com filtros por categoria e suporte a visualização em dispositivos móveis.

Backend: Uso de Spring Boot com controllers REST para a API e um Controller MVC para a renderização das páginas.

Regras de Negócio Customizadas: Implementação de lógica própria que não depende apenas da fonte de dados.
