# 🐉 TibiaBuilder 

> **Seu canivete suíço para o Tibia!** Um painel completo com simulador de set, enciclopédia de itens, fórum da comunidade e dashboard diário atualizado em tempo real.

## ⚔️ O que é isso?
Fala, galera! 👋 

O **TibiaBuilder** nasceu como um projeto pessoal focado em aprender e aplicar conceitos de desenvolvimento web, integração de APIs e Web Scraping, tudo feito de fã para fãs.

## ✨ O que já tá rolando? (Features)
- 📊 **Dashboard Diário:** Integração com a API do TibiaData v4. Mostra o Boss e a Criatura boostada do dia, além de rastrear automaticamente a localização do Rashid
- 📰 **Latest News em tempo real:** Puxa as últimas notícias do Tibia.com com opção de tradução automática via google translate direto no painel.
- 🛡️ **Set Builder:** Monte o seu personagem, equipe os itens e veja com o cálculo automático de status (armor, defesa, skills, bônus elementais no gráfico radar).
- 📚 **Enciclopédia:** Acervo completo puxado via Scraper. Filtre equipamentos por vocação, level e atributos.
- 💬 **Comunidade:** Um fórum interno com sistema de tópicos, respostas e *upvotes* para a galera trocar ideia.

## 🛠️ Tecnologias usadas

* **Back-end:** Java + Spring Boot (Spring MVC, Spring Security).
* **Front-end:** HTML5, Bootstrap 5, Javascript Vanilla, Chart.js (para o radar do set).
* **Template Engine:** Thymeleaf.
* **Integrações:** API TibiaData v4 e Web Scraping (TibiaWiki.com.br)
